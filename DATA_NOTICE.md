# Data Notice

This repository does not distribute WRDS, TAQ, or other licensed market data.

Code and configuration in this project may be used to request data from WRDS by authorized users under their institutional subscription and the applicable WRDS/vendor terms. Generated data files under `data/raw`, `data/interim`, `data/processed`, `data/cache`, and `outputs` are local artifacts and are ignored by Git.

Do not commit WRDS credentials, downloaded WRDS data, or files containing institution-specific access details.
